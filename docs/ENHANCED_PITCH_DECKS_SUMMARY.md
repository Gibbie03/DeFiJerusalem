All pitch decks use **ASCII-style diagrams** to clearly show:

### 1. Exact Placement
Shows where features appear in the UI:
```
┌─── HOMEPAGE ─────────────────────────┐
│ [Banner Here] ← Premium sponsor      │
│                                      │
│ ⭐ Featured Section                  │
│ ┌──────┐ ┌──────┐ ← Sponsored       │
│ │ Logo │ │ Logo │                   │
│ └──────┘ └──────┘                   │
└──────────────────────────────────────┘
```

### 2. Visual Styling
Shows badges, borders, colors, icons:
```
✓ = Verified badge (blue)
🌟 = Featured (yellow)
⭐ = Premium (gold)
SPONSORED = Purple badge
```

### 3. User Experience
Shows what users see and how they interact:
```

[Logo] = Protocol branding
Score: 95/100 = Security display
```
