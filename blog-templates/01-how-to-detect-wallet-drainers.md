### Section 2: 10 Warning Signs of Wallet Drainers (500 words)

**Create a numbered list**:

1. **Suspicious Domain Names**
   - .lol, .tk, .ml, .ga domains (90%+ scam rate)
   - Typosquatting (uniswaap.com, aave-cliam.org)
   - Example: "register-metamask.net"

2. **Urgent "Limited Time" Offers**
   - "Claim your free airdrop NOW!"
   - "Eligible for 1000 UNI tokens - 24 hours only"
   - Creates panic, bypasses logic

3. **Unrealistic APY Promises**
   - "Guaranteed 1000% APY risk-free!"
   - Context matters: High APY + "guaranteed" = scam
   - Legitimate farms can have high APY but never "guaranteed"

4. **Unsolicited DMs on Twitter/Telegram**
   - Fake support accounts
   - "Admin" impersonators
   - Never trust unsolicited messages

5. **Requests for Seed Phrases/Private Keys**
   - NEVER legitimate
   - No real protocol asks for this
   - Instant red flag

6. **Multiple Approval Requests**
   - Single transaction requesting approval for all tokens
   - "Permit2" abuse (legitimate tool used maliciously)
   - Check what you're signing!

7. **Pixel-Perfect Website Clones**
   - Looks exactly like Uniswap/Aave
   - Wrong domain
   - Use official links only

8. **No Social Media Presence**
   - Real protocols have Twitter, Discord, GitHub
   - Check follower counts (can't fake 100k+ real followers)

9. **Fake Security Audits**
   - Made-up audit firms
   - No verifiable audit report
   - Check JERUSALEM scanner for real audits

10. **Seaport.js or Unknown Contract Calls**
    - Technical red flag
    - Drainer infrastructure
    - Reject if you see unfamiliar contracts
